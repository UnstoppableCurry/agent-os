# 行为模式

> 最后更新: 2026-03-03
> 记录经过多次观察确认的行为模式

---

## 正向模式 (保持)

### 先锋红利捕获
- **触发**: 新技术/工具出现
- **行为**: 第一时间体验 -> 快速评估价值 -> 决定是否深入
- **结果**: 比大多数人早获得经验和认知
- **保持方式**: 设定认知投资预算，有意识地做

### 痛点驱动创造
- **触发**: 发现自己或他人的真实痛点
- **行为**: 爆发毅力 -> 熬夜也要解决 -> 直到产出
- **结果**: 有价值的产品/工具/解决方案
- **保持方式**: 始终从痛点出发，不从技术出发

### 深度追问
- **触发**: 遇到复杂问题
- **行为**: 追问"为什么"到最底层 -> 不满足于表面答案
- **结果**: 建立深层认知架构，解决根本问题
- **保持方式**: 不急于求快，允许深度思考时间

## 需要觉察的模式

### 情绪消费循环
- **触发**: 焦虑、空虚、无聊、嫉妒
- **行为**: 冲动购买 -> 短暂满足 -> 更深的空虚
- **打断方式**:
  1. 觉察情绪: "我现在真正需要什么？"
  2. 替代行为: 去创造/运动/发内容
  3. 延迟机制: 购物车等 3 天

### 项目频繁切换
- **触发**: 新项目的新鲜感消退 + 看不到价值连接
- **行为**: 放下当前 -> 开始新的 -> 循环
- **打断方式**:
  1. 启动前检查: "做成了谁受益？"
  2. 控制并行项目 <= 3 个
  3. 设置阶段性里程碑，小步验证价值

### 熬夜透支
- **触发**: 进入心流状态 + 对项目有激情
- **行为**: 深夜编码 -> 忽略睡眠 -> 次日状态差
- **打断方式**:
  1. 设置硬性截止时间
  2. 把"灵感"记录下来而不是立刻执行
  3. 记住: 持续输出 > 一次爆发

### 比较心理
- **触发**: 看到他人的成功/财富/自由
- **行为**: 嫉妒 -> 自我怀疑 -> 焦虑 -> 可能触发情绪消费
- **打断方式**:
  1. 重新解读: 嫉妒是路标，不是否定
  2. 回到自己的节奏和路径
  3. 看自己的进度（对比过去的自己，而非他人）

## 决策模式

### 高效路径
探索(先锋) -> 价值评估(连接器) -> 深度投入(建造者) -> 展示(表演者) -> 反馈(感受器)

### 失败路径
技术自嗨 -> 看不到价值 -> 动力断裂 -> 放弃 -> 自责("半途而废")

### 关键检查点
1. 启动前: 价值连接清晰吗？
2. 进行中: 有阶段性的展示和反馈吗？
3. 疲惫时: 是该休息还是该坚持？
4. 放弃前: 是价值确实不存在，还是只是遇到了困难？
