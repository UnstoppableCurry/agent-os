# 时间管理

> 最后更新: 2026-03-03

---

## 典型日程模式

| 时段 | 活动 | 备注 |
|------|------|------|
| 上午 | 工作 (动码印章) | 算法工程师 |
| 下午 | 工作 (动码印章) | |
| 晚上 | 副业/学习/个人项目 | 主要创造时间 |
| 深夜 | 编码（熬夜） | 容易进入心流但影响健康 |

---

## 时间分配目标

| 领域 | 目标占比 | 当前估算 |
|------|----------|----------|
| 主业工作 | 40% | ~45% |
| 个人项目 | 25% | ~20% |
| 学习成长 | 15% | ~10% |
| 健康运动 | 10% | ~5% |
| 社交关系 | 10% | ~10% |
| 休息恢复 | - | 不足 |

---

## 时间使用原则

1. 深度工作优先: 不被碎片化打断的创造时间最珍贵
2. 早期投入回报高: 新技术/工具的先锋红利有时间窗口
3. 熬夜是借贷: 偶尔可以，但不能成为常态
4. 运动时间不可省: 回血活动是创造力的燃料

## 理想的一天 (来自天赋说明书)

> 在某个美好的地方，做着热爱的创造性工作，
> 中途去运动回血，
> 录制内容向粉丝传递价值，
> 最后与家人/伴侣享受世界的美好。

---

## 时间陷阱警觉

- 情绪消费导致的时间浪费（刷手机、冲动购物）
- 项目频繁切换导致的上下文切换成本
- 技术自嗨（研究不会用到的技术）
