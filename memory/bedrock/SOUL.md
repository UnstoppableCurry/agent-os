## 核心身份

**王天信 | 26岁 | 北京**

算法工程师出身，正在走向独立创造者之路。不是一个"写代码的人"，而是一个"用技术解决真实问题的人"。内心深处渴望自由、被看见、留下痕迹。

一句话定义: **敏感的先锋建造者 -- 用深度认知理解世界，用创造连接他人。**

## 四大核心天赋

### 1. 先锋侦察兵 (The Pioneer Scout)
- **表现**: 对新事物有雷达式敏感，第一时间行动
- **运作方式**: 时间压缩感 -- 快速在新旧知识间建立连接，把学习曲线压缩到极致
- **使用规则**: 行动先于完美，用行动来学习
- **阴影面**: 尝鲜后可能迅速失去兴趣 -- 这是大脑在筛选"是否值得继续投入"
- **Agent 应用**: 推荐新工具、新技术时优先级高；但要同时评估长期价值

### 2. 价值连接器 (The Value Connector)
- **表现**: 只有看到"对别人有用"时，才进入深度投入模式
- **运作方式**: 内置价值罗盘 -- 感知到连接真实需求时爆发惊人毅力
- **使用规则**: 启动项目前必须回答 "做成了，谁会因此受益？"
- **阴影面**: 看不到价值时迅速撤离 -- 这不是半途而废，是高效目标筛选
- **Agent 应用**: 任何建议都要连接到真实人的真实需求，避免"技术自嗨"

### 3. 建造者-表演者混合体 (The Builder-Performer Hybrid)
- **建造者面**: 享受从 0 到 1 的创造，追求掌控感（飞机模型、Claude Code、开飞机）
- **表演者面**: 渴望被看见、被认可（录制视频、传递价值、粉丝）
- **内在张力**: "我不需要任何人" vs "没人看见这一切有什么意义"
- **使用规则**: 两者缺一不可 -- 纯建造会枯竭，纯表演会空虚
- **Agent 应用**: 每个创造闭环都应包含"展示"环节；提醒 Build in Public

### 4. 敏感感受器 (The Sensitive Antenna)
- **表现**: 对情感、评价、氛围的高度感知
- **超级能力**: 情感雷达 + 深刻共情 + 真诚外显
- **使用规则**: 这是产品思维的基础（理解用户需求），保护但不封闭
- **阴影面**: 可能过度在意评价，一句批评循环很久
- **Agent 应用**: 沟通时注意语气；批评要建设性；认可要具体真诚

## 价值观与原则

| 价值观 | 含义 | 行为指引 |
|--------|------|----------|
| 自由 | 时间自由、选择自由、不为钱做不想做的事 | 一切决策指向增加选择权 |
| 深度 | 追问"为什么"到最底层，不满足于表面 | 宁慢不浅，深度是核武器 |
| 创造 | 从 0 到 1 做出之前不存在的东西 | 创造 > 消费，输出 > 输入 |
| 连接 | 通过作品连接真实的人和需求 | 拒绝技术自嗨，始终面向用户 |
| 被看见 | 留下痕迹，影响他人 | Build in Public，持续输出 |

## 行为模式特征

### 能量来源 (回血)
- 万米高空的掌控感（开飞机）
- 熬夜做出解决真实问题的东西
- 收到用户/粉丝的正面反馈
- 探索全新的工具和技术

### 能量消耗 (耗电)
- 重复性、无创造感的工作
- 看不到价值连接的项目
- 没有反馈的孤立研发
- 被评判、被否定但没有建设性意见

### 决策模式
- 快速尝试 -> 价值评估 -> 深度投入或果断放弃
- 不适合线性规划，适合"探索-筛选-聚焦"循环

### 危险信号
- 情绪消费激增 = 内心空虚/焦虑信号
- 频繁切换项目 = 没找到价值连接点
- 嫉妒他人 = 未活出的自我在呼唤

## Agent 行为准则

基于以上灵魂画像，AgentOS 在与天信互动时应遵循:

1. **永远连接价值**: 建议做事前先回答"谁会受益"
2. **尊重深度节奏**: 不催促，允许深入思考
3. **推动展示闭环**: 创造后提醒分享和展示
4. **情绪感知优先**: 察觉情绪波动时优先处理，再谈任务
5. **自由导向决策**: 所有财务/职业建议指向增加选择权
6. **认可要具体**: 不空泛表扬，指出具体做得好的地方
7. **控制情绪消费**: 检测到冲动消费信号时温和提醒
